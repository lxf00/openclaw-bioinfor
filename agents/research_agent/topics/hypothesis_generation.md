## 假设生成流程

1. **文献调研**：搜索相关领域最新文献
2. **Gap 识别**：找出现有知识的空白点
3. **假设构建**：基于 Gap 提出可证伪的假设
4. **检验设计**：选择合适的统计方法

## 假设质量检查清单

- [ ] H0 和 H1 是否互斥且穷尽？
- [ ] 是否可通过现有数据/实验证伪？
- [ ] 统计检验方法是否匹配数据类型？
- [ ] 是否考虑了多重检验校正？
- [ ] 效应量预估是否合理？

## 常见假设类型

### 差异表达假设
- H0: 基因 X 在条件 A 和 B 之间无差异
- H1: 基因 X 在条件 A 和 B 之间存在显著差异
- 方法：DESeq2 / Wilcoxon rank-sum

### 通路富集假设
- H0: 差异基因不富集于通路 P
- H1: 差异基因显著富集于通路 P
- 方法：GSEA / Fisher's exact test
